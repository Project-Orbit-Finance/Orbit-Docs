# Orbit-Docs

Repositório oficial de documentação do projeto Orbit.

## Estrutura do ecossistema

O projeto está dividido em três repositórios na organização do GitHub:

* `Orbit-Docs` - documentação, backlog, ADRs e contratos oficiais
* `Orbit-Finance-Web` - frontend da aplicação
* `Orbit-Finance-Api` - backend Laravel e persistência

## Propósito deste repositório

Este repositório centraliza a referência oficial do produto e da engenharia do Orbit.
Ele não contém a implementação da aplicação.
